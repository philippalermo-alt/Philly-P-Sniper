# NHL Totals V2 KPI Specification

## Overview
Daily Operations Reporting schema (`YYYY-MM-DD.json`) generated by `scripts/ops/generate_nhl_kpi.py`.

## Schema
```json
{
  "date": "YYYY-MM-DD",
  "timestamp": "ISO8601",
  "status": "OK|ERROR",
  
  "scans_count": "integer (pipeline iterations)",
  "eligible_count": "integer (games with lines)",
  "predictions_count": "integer (games successfully modeled)",
  "recommendations_count": "integer (bets with EV > Threshold)",
  
  "avg_ev_recommended": "float (0.05 = 5%)",
  
  "sigma_current": "float (Fixed: 2.2420)",
  
  "error": "string (optional)"
}
```

## Alert Logic
*   **Data Join Failure**: `predictions_count < (eligible_count * 0.8)`
*   **Excessive Action**: `recommendations_count > 10` (Warning)
*   **Sigma Drift**: `sigma_current != 2.2420` (Critical - Config Change)
